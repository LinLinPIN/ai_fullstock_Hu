    - props + emits 
        子组件不可以直接修改props，vue 数据除v-model 都是单向的
        单向可以确保数据的正确性
        如果要修改:emit
            - 父组件会以自定义事件的方式，定义一个事件来修改自己的值并接受传参同步00状态
            - 子组件通过 defineEmits 拿到这个方法传入参数就能修改父组件的值了
